Thank you for purchasing and using Jamroom 5 and the PhotoPro skin.

If you've installed Jamroom 5, or had Jamroom 5 installed by a Jamroom 5 staff, for the first time, the jrPhotoPro skin will not be the default skin. This skin can be activated via the skins info tab.

Or if you prefer not to install the Jamroom 5 PhotoPro skin yourself, we have a guaranteed installation service available, visit us for more info: http://www.jamroom.net for more info.

Skin Activation:

To activate the PhotoPro skin for your Jamroom5 Install, after uploading the skin to your skins directory with a FTP program or via the marketplace, log-in as the site admin and click the Skins tab, then click on the PhotoPro skin from the menu entry, then from the Info tab section check the "Set As Active Skin" checkbox and click the save button.


Site branding:

Simply edit or replace the following files to change the branding of your site.

  • Main Header Logo (site wide | width: 236px height: 55px) - skins/jrPhotoPro/img/logo.png
  • Footer Information - skins/jrPhotoPro/footer.tpl

Note: You may also modify skin template via the "Templates" tab in the admin skins section of the control panel.


PhotoPro Settings (skins/jrPhotoPro/config.php file):

PhotoPro has a couple of configuration settings, as admin you will see a link in the ACP drop down menu which links to the PhotoPro Skin Settings page in your admin.

Note: Please take your time to view the PhotoPro Settings and descriptions.


Slider Quota ID's:

You can choose which of your quotas will be allowed to show in the Slider/Carousel. You can manually select which Quotas wil be allowed show in the slider by adding the Quota ID(s) separated by a comma ( ie. 1,2,3 etc. ) in the Image Quota(s) field of the PhotoPro Settings.

Note: If you leave this field blank, the Slider will show images from all quotas.



Slider Image ID's:

You can manually select which Gallery Images to show in the slider by adding the Gallery Photo ID separated by a comma ( ie. 1,2,3 etc. ) in the Image ID(s) field of the PhotoPro Settings.

Note: If you leave this field blank, the Slider will random images.


Features:
  • Responsive
  • Swipe scroll enabled. (For Mobile Devices)
  • Mouse scroll enabled. (For PC)
  • Hiding navigation arrows for both the slider and carousel.
  • Image Slider and Carousel are synced together.

Featured Gallery:

Enter the Gallery Title you want to show in the Featured Gallery image slider.

Note: If this field is left blank the image slider will show all Galleries.


Site Ads/Google Ads:

Google Ads have been configured throughout the templates, your Google Ads code for ALL ads needs to be set in PhotoPro Settings.

If you prefer to use the two ad fields, set the Google Ads to NO in PhotoPro Settings. Check the Site Ads check box to hide all ads on the site.

Note: As of now, the Jamroom 5 does NOT have an Ad Manager Module.


Require Images:

If you prefer to not show images or profiles that have not uploaded an image, even though an entry was created, check this setting and only entries/profiles with images will show.


Hide Tag Cloud:

If you prefer to not show the tag cloud on your index page, check this setting and tag cloud on the index page will be hidden.

Note: This will not hide the tag cloud on the profile pages.


Flickr Feed:

jrPhotoPro is setup to display a Flickr Feed. To show this feed, you'll need to have the jrFeed module installed and activated.

In the tools section of the Feed module, click the create/update feed button

Then add the following to the feed name and feed url fields.

  • Feed Name: flickr_feed
  • Feed URL: https://api.flickr.com/services/feeds/photos_public.gne?format=rss2

Then click the Create button.

Note: If the Feed module is not active, then the site will show recent comments. If the Feed module is active and you have not created the Flickr Feed as described above, then the feed will show Jamroom's Facebook feed.


Language Files:

Although the jrPhotoPro skin is not setup for multiple languages, it can be configured for Jamroom's multilingual system which allows the site language to be quickly changed by clicking on a flag image. If you wish to use an additional language for your site you can do so by translating from the existing language file which is located in the "skins/jrPhotoPro/lang/" directory. Once you have completed the translation, save the file as xx-XX.php with "xx" being your countries suffix in lower case and XX being your countries suffix in upper case. You will then need to edit the file skins/jrPhotoPro/header.tpl, or the template where you would like to place the link to change the language, and add the following line:

Again changing xx-XX and xx.png to your countries suffix in lower case and upper case. Also change LANGUAGE to the new language, ie. Spanish etc...
